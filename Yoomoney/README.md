# Плагин ЮMoney для Xboard

Этот плагин позволяет принимать платежи в Xboard через HTTP-уведомления (переводы на кошелек для физических лиц) от ЮMoney (ранее Яндекс.Деньги). **Это решение предназначено для обычных кошельков, а не для ЮKassa.**

## Возможности
- Оплата через форму `quickpay`.
- Автоматическое подтверждение платежа (через HTTP-уведомления).
- Валидация подписи (`sha1_hash`) с использованием секретного слова.

## Предварительные требования
1. У вас должен быть активированный и идентифицированный кошелек [ЮMoney](https://yoomoney.ru/).
2. Базовая валюта вашего Xboard должна быть настроена как Рубли (RUB), или вы должны учитывать, что ЮMoney принимает и обрабатывает суммы только в рублях. Плагин передает сумму заказа в рублях (делит значение в базе `total_amount` на 100).

## Инструкция по настройке ЮMoney

1. Авторизуйтесь в своем кошельке [ЮMoney](https://yoomoney.ru/).
2. Перейдите в раздел **"Настройки"** -> вкладка **"Пакеты/Другие сервисы"** -> блок **"HTTP-уведомления"** (или перейдите по прямой ссылке [https://yoomoney.ru/transfer/myservices/http-notification](https://yoomoney.ru/transfer/myservices/http-notification)).
3. В поле **URL для уведомлений** введите адрес обработчика Xboard. Обычно он выглядит так:
   ```
   https://ваш-домен.com/api/v1/guest/payment/notify/Yoomoney/ваш_уникальный_uuid_метода
   ```
   > **Важно:** Точный `URL для уведомлений` вы сможете скопировать в панели администратора Xboard после создания метода оплаты (кнопка копирования возле метода в списке).
4. Нажмите **"Показать секрет"** и скопируйте **Секретное слово**. Оно понадобится для настройки плагина.
5. Поставьте галочку **"Отправлять HTTP-уведомления"** и нажмите **"Готово"**.

## Инструкция по настройке Xboard

1. Скопируйте папку `Yoomoney` в директорию `plugins/` (а **не** в `app/Payments/`) в корневом каталоге вашего проекта Xboard.
   *(Новые плагины с пространством имен `Plugin\` должны располагаться именно в папке `plugins/` для корректной автозагрузки).*
2. Перейдите в **Панель администратора Xboard**.
3. Откройте раздел **Настройки** -> **Методы оплаты** (Payment Methods).
4. Нажмите **"Добавить метод оплаты"**.
5. Выберите плагин **Yoomoney**.
6. Заполните поля конфигурации:
   - **Отображаемое имя:** например, `ЮMoney`.
   - **Иконка (URL):** URL логотипа ЮMoney (по умолчанию используется официальный логотип).
   - **Номер кошелька ЮMoney:** номер вашего кошелька (начинается с `41001...`).
   - **Секретное слово:** секретное слово, которое вы скопировали в настройках HTTP-уведомлений ЮMoney.
7. Сохраните настройки и включите метод оплаты.
8. После сохранения скопируйте **Notify URL (URL для уведомлений)** из списка методов оплаты и вставьте его в настройки ЮMoney (см. пункт 3 предыдущего раздела).

## Важные примечания
- Плагин формирует форму оплаты, которая автоматически (с помощью JavaScript) отправляет пользователя на страницу оплаты ЮMoney.
- ЮMoney присылает HTTP-уведомление только в случае **успешной** оплаты.
- Обязательно проверьте, что ваш сервер (где установлен Xboard) имеет открытый доступ к интернету для приема POST-запросов от серверов ЮMoney.
- В настройках ЮMoney нельзя указать динамический `Return URL` (страницу возврата после оплаты). Плагин передает `successURL` в форме оплаты, по которому пользователь будет перенаправлен обратно в магазин после успешной транзакции на стороне ЮMoney.
