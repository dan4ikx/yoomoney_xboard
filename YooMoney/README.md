# YooMoney (ЮMoney) P2P Payment Plugin for Xboard

Этот плагин позволяет принимать платежи в Xboard через переводы на кошелек ЮMoney (бывший Яндекс.Деньги) с помощью формы Quickpay и HTTP-уведомлений.

## Установка

1. Скопируйте папку `YooMoney` в директорию `plugins/` вашего Xboard. Убедитесь, что название папки в точности `YooMoney` (с большой буквой **M**). У вас не должно быть других папок с плагинами ЮMoney (например, `Yoomoney` с маленькой `m` или старых файлов в `app/Payments/`), чтобы избежать конфликтов (Fatal error).
2. Зайдите в контейнер и очистите кэш (обязательно):
   ```bash
   docker exec -it <имя_контейнера_xboard> php artisan optimize:clear
   ```
   *(либо `docker compose exec xboard php artisan optimize:clear`, если используете docker compose)*
3. Перейдите в админ-панель Xboard -> Настройки -> Управление плагинами. Убедитесь, что плагин YooMoney включен.
4. Перейдите в "Способы оплаты" и добавьте новый способ оплаты, выбрав шлюз **YooMoney**.

## Настройка ЮMoney (Важно!)

Для того чтобы Xboard автоматически зачислял средства после оплаты, нужно правильно настроить HTTP-уведомления (вебхуки) в вашем кошельке ЮMoney.

Xboard **скрывает** уникальный идентификатор (UUID) способа оплаты в админке. В поле "Домен уведомлений" (Notification Domain) Xboard покажет вам ссылку вида:
`https://ВАШ_ДОМЕН/api/v1/guest/payment/notify/YooMoney`
**Эта ссылка НЕ БУДЕТ РАБОТАТЬ!** Xboard будет выдавать ошибку `payment not found`.

Чтобы получить **правильную** ссылку с уникальным хэшем для вебхука, выполните следующие действия:

1. В панели Xboard создайте новый способ оплаты YooMoney (или используйте существующий) и заполните номер кошелька ЮMoney и секретное слово (пока можно указать любое слово, позже мы его заменим).
2. Как обычный пользователь, создайте тестовый заказ на любую сумму и нажмите кнопку "Оплатить". Вас перекинет на сайт ЮMoney.
3. Откройте логи Xboard на вашем сервере. Если вы используете Docker:
   ```bash
   docker exec -it <имя_контейнера_xboard> tail -n 50 /www/storage/logs/laravel.log
   ```
4. Найдите в логах строку:
   `local.INFO: YooMoney EXACT NOTIFY URL FROM XBOARD: {"url":"https://ВАШ_ДОМЕН/api/v1/guest/payment/notify/YooMoney/xxxxxxxx"}`
5. Скопируйте **ПОЛНУЮ** ссылку (вместе с хэшем на конце `xxxxxxxx`).
6. Перейдите в свой кошелек ЮMoney: **Настройки -> Пакеты / Другие настройки -> HTTP-уведомления** (или по прямой ссылке [https://yoomoney.ru/transfer/myservices/http-notification](https://yoomoney.ru/transfer/myservices/http-notification)).
7. Вставьте скопированную ссылку в поле "URL для уведомлений". Поставьте галочку "Отправлять HTTP-уведомления".
8. Нажмите "Показать секрет" и скопируйте "Секретное слово".
9. Вернитесь в админ-панель Xboard, откройте настройки способа оплаты YooMoney и вставьте это "Секретное слово" в соответствующее поле.

Всё готово! Теперь платежи будут обрабатываться корректно, а пустые поля (`null`) при пополнении с карты будут правильно проверяться плагином без ошибки "Missing required fields".

> **Внимание:** НИКОГДА не удаляйте способ оплаты YooMoney в панели Xboard после того, как вы его настроили. Если вы его удалите и создадите заново, Xboard сгенерирует **новый уникальный хэш**, и вам придется повторять шаги с 1 по 9 для получения новой ссылки вебхука.
