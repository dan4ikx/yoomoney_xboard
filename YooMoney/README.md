# YooMoney Plugin для XBoard

Плагин ЮMoney (P2P-переводы, не ЮKassa) позволяет вам принимать оплату напрямую на ваш кошелек ЮMoney от пользователей. Оплата работает через форму Quickpay, а проверка платежей осуществляется автоматически через HTTP-уведомления.

## Установка

1. Скопируйте папку `YooMoney` в каталог `plugins/` вашего проекта XBoard.
2. В админ-панели XBoard перейдите в раздел "Способы оплаты" или "Плагины".
3. Включите плагин **YooMoney**.

## Настройка на стороне ЮMoney

1. Зарегистрируйте и идентифицируйте кошелек ЮMoney (если еще этого не сделали).
2. Перейдите на страницу настройки **HTTP-уведомлений**: [https://yoomoney.ru/transfer/myservices/http-notification](https://yoomoney.ru/transfer/myservices/http-notification)
3. В поле **URL для уведомлений** вставьте URL, который указан в XBoard на странице настройки способа оплаты (обычно он выглядит так: `https://ваш-домен/api/v1/guest/payment/notify/YooMoney/УНИКАЛЬНЫЙ_ИДЕНТИФИКАТОР`). **Внимание:** Скопируйте точный URL из панели Xboard, иначе вы увидите ошибку "payment not found".
4. Нажмите кнопку **Показать секрет** и скопируйте "Секретное слово" — оно понадобится в настройках XBoard.
5. Отметьте галочкой "Отправлять HTTP-уведомления" и сохраните настройки.

## Настройка в XBoard

В настройках плагина YooMoney в панели администратора заполните следующие поля:

- **Отображаемое имя (Display Name)**: ЮMoney (или как вы хотите назвать способ оплаты для клиентов).
- **Номер кошелька ЮMoney (Receiver)**: Ваш номер кошелька, который начинается с `41001...`
- **Секретное слово (Secret)**: Секретное слово, которое вы скопировали из настроек HTTP-уведомлений ЮMoney.

## Примечания
- Баланс в Xboard ведется в валюте, которую вы установили по умолчанию (для этого плагина это должны быть рубли, так как ЮMoney оперирует рублями).
- В данном плагине используется передача номера заказа в поле `label`, чтобы автоматически зачислять платеж пользователю.
- Зачисление средств в биллинге происходит по отправленной пользователем сумме (без учета комиссии ЮMoney). Таким образом, платеж в биллинге будет успешно подтвержден даже с учетом комиссии системы.
