# Плагин оплаты ЮMoney для Xboard

Этот плагин позволяет добавить метод оплаты через кошелек ЮMoney (бывший Яндекс.Деньги) в ваш биллинг Xboard. Для работы используется форма «Быстрый платеж» (Quickpay) и HTTP-уведомления ЮMoney.

## Особенности
* Оплата банковскими картами или из кошелька ЮMoney
* Автоматическая проверка зачисления средств с помощью проверки sha1 хэша
* Поддержка возврата пользователя после оплаты (редирект)
* Не требует юридического лица (работает на обычном или идентифицированном кошельке физического лица)
* Совместимость с новой архитектурой плагинов Xboard (`plugins/`)

## Инструкция по установке

### Шаг 1: Загрузка плагина
Скачайте папку `Yoomoney` (содержащую файлы `Plugin.php` и `config.json`) и загрузите её на ваш сервер с Xboard в директорию `plugins`.
Структура файлов должна выглядеть так:
```
/путь/до/вашего/xboard/plugins/Yoomoney/Plugin.php
/путь/до/вашего/xboard/plugins/Yoomoney/config.json
```
*Убедитесь, что права на папку и файлы позволяют серверу читать их (например, 755 для директорий и 644 для файлов).*

Альтернативный вариант: упакуйте папку `Yoomoney` в `.zip` архив и загрузите его через панель администратора Xboard (раздел Управление плагинами -> Загрузить).

### Шаг 2: Установка и активация
1. Войдите в панель администратора Xboard.
2. Перейдите в раздел плагинов (Plugins) и найдите `YooMoney` в списке.
3. Нажмите кнопку **Установить** (Install), а затем **Активировать** (Enable).

### Шаг 3: Настройка кошелька ЮMoney
1. Перейдите в кошелек ЮMoney и авторизуйтесь.
2. Откройте настройки HTTP-уведомлений по адресу: [https://yoomoney.ru/transfer/myservices/http-notification](https://yoomoney.ru/transfer/myservices/http-notification)
3. Нажмите "Показать секрет" и скопируйте ваше **Секретное слово**. (Оно понадобится для настройки плагина).
4. Укажите URL для уведомлений, который вам выдаст панель управления Xboard после создания платежного шлюза (см. Шаг 4).
5. Поставьте галочку "Отправлять HTTP-уведомления" и сохраните настройки.

### Шаг 4: Добавление шлюза в методы оплаты Xboard
1. В панели администратора Xboard перейдите в раздел **Платежи** (Payments) -> **Способы оплаты** (Payment Methods).
2. Нажмите кнопку **Добавить** (Add).
3. Заполните данные:
   * **Название**: ЮMoney (или любое другое, которое увидят пользователи)
   * **Платежный шлюз**: выберите `YooMoney` (или `yoomoney`) в выпадающем списке.
   * **Номер кошелька ЮMoney**: укажите ваш номер кошелька (обычно состоит из 15 цифр и начинается с 4100).
   * **Секретное слово**: вставьте секретное слово, полученное на Шаге 3.
4. Сохраните изменения и включите созданный способ оплаты.
5. Откройте только что созданный шлюз в панели и скопируйте `URL уведомления` (Notify URL) и вставьте его в настройки ЮMoney (Шаг 3, пункт 4).

## Безопасность
* Плагин использует хэширование `sha1` для проверки подлинности входящих HTTP-уведомлений от ЮMoney.
* Плагин проверяет сумму платежа до вычета комиссии, чтобы гарантировать полную оплату заказа.
* Переводы с кодом протекции игнорируются плагином, пока не будут подтверждены, во избежание мошенничества.

## Поддержка
Для нормальной работы шлюза ваш сервер Xboard должен быть доступен извне и иметь валидный SSL-сертификат (HTTPS), так как ЮMoney отправляет уведомления только на защищенные URL.
