# Плагин оплаты ЮMoney для Xboard

Этот плагин позволяет добавить метод оплаты через кошелек ЮMoney (бывший Яндекс.Деньги) в ваш биллинг Xboard (а также V2Board). Для работы используется форма «Быстрый платеж» (Quickpay) и HTTP-уведомления ЮMoney.

## Особенности
* Оплата банковскими картами или из кошелька ЮMoney
* Автоматическая проверка зачисления средств с помощью проверки sha1 хэша
* Поддержка возврата пользователя после оплаты (редирект)
* Не требует юридического лица (работает на обычном или идентифицированном кошельке физического лица)

## Инструкция по установке

### Шаг 1: Загрузка файлов
Скачайте файлы `YooMoney.php` и `YooMoney.json` из папки `app/Payments` и загрузите их на ваш сервер с Xboard по следующему пути:
```
/путь/до/вашего/xboard/app/Payments/YooMoney.php
/путь/до/вашего/xboard/app/Payments/YooMoney.json
```
*Убедитесь, что права на файлы позволяют серверу читать их (например, 644).*

### Шаг 2: Настройка кошелька ЮMoney
1. Перейдите в кошелек ЮMoney и авторизуйтесь.
2. Откройте настройки HTTP-уведомлений по адресу: [https://yoomoney.ru/transfer/myservices/http-notification](https://yoomoney.ru/transfer/myservices/http-notification)
3. Нажмите "Показать секрет" и скопируйте ваше **Секретное слово**. (Оно понадобится для настройки плагина).
4. Укажите URL для уведомлений, который вам выдаст панель управления Xboard после создания платежного шлюза (см. Шаг 3).
5. Поставьте галочку "Отправлять HTTP-уведомления" и сохраните настройки.

### Шаг 3: Добавление шлюза в Xboard
1. Войдите в панель администратора Xboard.
2. Перейдите в раздел **Платежи** (Payments) -> **Способы оплаты** (Payment Methods).
3. Нажмите кнопку **Добавить** (Add).
4. Заполните данные:
   * **Название**: ЮMoney (или любое другое, которое увидят пользователи)
   * **Платежный шлюз**: выберите `YooMoney` в выпадающем списке (он появится автоматически после загрузки файла)
   * **Номер кошелька ЮMoney**: укажите ваш номер кошелька (обычно состоит из 15 цифр и начинается с 4100)
   * **Секретное слово**: вставьте секретное слово, полученное на Шаге 2.
5. Сохраните изменения.
6. Включите созданный способ оплаты.
7. Откройте только что созданный шлюз в панели и скопируйте `URL уведомления` (Notify URL) и вставьте его в настройки ЮMoney (Шаг 2, пункт 4).

## Безопасность
* Плагин использует хэширование `sha1` для проверки подлинности входящих HTTP-уведомлений от ЮMoney.
* Переводы с кодом протекции игнорируются плагином, пока не будут подтверждены, во избежание мошенничества.

## Поддержка
Для нормальной работы шлюза ваш сервер Xboard должен быть доступен извне и иметь валидный SSL-сертификат (HTTPS), так как ЮMoney отправляет уведомления только на защищенные URL.
